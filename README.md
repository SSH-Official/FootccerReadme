# ProjectFootccer
#### 프로젝트명
Footccer 프로그램 프로젝트
#### 개요
축구, 풋살 등의 활동을 같이할 사람을 매칭해주는 어플리케이션입니다.
(대구인자위 프로젝트 발표회 출품 프로젝트)
#### 개발기간
2023년 9월 1일 ~ 2023년 10월 26일
#### 멤버구성
성승현, 편성진, 이창민, 이철민, 박윤환, 이해준
